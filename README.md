# Analizador de Quiniela

Programa local para cargar resultados de la Tombola Santiagueña, consultar numeros por fecha, turno, posicion y terminaciones, y combinar datos con numeros de los sueños.

## Como ejecutarlo

1. Instalar Node.js si no esta instalado.
2. Abrir una terminal en esta carpeta.
3. Ejecutar:

```bash
npm start
```

4. Abrir:

```text
http://localhost:3000
```

## Datos

Los resultados se guardan en `data/results.json`.

## Importante

Este programa sirve para entretenimiento y analisis de resultados historicos. Los sorteos son de azar y no garantiza predicciones.
